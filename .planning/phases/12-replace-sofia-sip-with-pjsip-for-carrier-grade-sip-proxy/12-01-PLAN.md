Create the install script and the full pjsip-sys crate as specified in the design doc (docs/plans/2026-04-01-pjsip-migration.md), Phase 1, Tasks 1.1 and 1.2.

1. Create `scripts/install-pjproject.sh`:
   - Downloads pjproject 2.14.1 from GitHub
   - Configures with --disable-video --disable-sound --enable-shared
   - macOS OpenSSL fallback (--with-ssl=/usr/local/opt/openssl, retry without)
   - PREFIX env var override (default /usr/local)
   - Make it executable (chmod +x)

2. Create `crates/pjsip-sys/Cargo.toml`:
   - name = "pjsip-sys", edition = "2024", links = "pjsip"
   - build-dependencies: bindgen = "0.71", pkg-config = "0.3"
   - dependencies: libc = "0.2"

3. Create `crates/pjsip-sys/pjsip_wrapper.h`:
   - Include pjlib base (pj/types.h, pj/pool.h, pj/log.h, pj/os.h, pj/string.h, pj/timer.h, pj/errno.h)
   - Include pjlib-util DNS (resolver.h, srv_resolver.h, dns.h)
   - Include pjsip core (sip_transport.h, sip_transaction.h, sip_endpoint.h, sip_module.h, sip_event.h, sip_msg.h, sip_uri.h, sip_auth.h, sip_dialog.h, sip_ua_layer.h, sip_util.h, sip_resolve.h)
   - Include pjsip-ua (sip_inv.h, sip_regc.h, sip_replaces.h, sip_xfer.h, sip_100rel.h, sip_timer.h)
   - Include pjsip-simple (evsub.h)
   - Include pjmedia SDP only (sdp.h, sdp_neg.h) — NO audio device headers

4. Create `crates/pjsip-sys/build.rs`:
   - pkg-config probe for "libpjproject" (atleast_version "2.14")
   - Emit link search paths and lib names from pkg-config
   - bindgen::Builder with comprehensive allowlist (see design doc Task 1.2 Step 3 for full list):
     - pjlib: pj_init, pj_shutdown, pj_pool_*, pj_caching_pool_*, pj_thread_*, pj_log_*, pj_str, pj_strerror
     - pjsip endpoint: pjsip_endpt_*
     - pjsip transport: pjsip_udp_transport_start*, pjsip_tcp_transport_start*, pjsip_tls_transport_start*
     - pjsip transaction: pjsip_tsx_*
     - pjsip module: pjsip_endpt_register/unregister_module
     - pjsip message types: pjsip_msg*, pjsip_hdr*, pjsip_generic_string_hdr*, etc.
     - pjsip-ua dialog: pjsip_dlg_*
     - pjsip-ua INVITE session: pjsip_inv_*
     - pjsip-ua session timer: pjsip_timer_*
     - pjsip-ua 100rel: pjsip_100rel_*
     - pjsip-ua replaces: pjsip_replaces_*
     - pjsip-ua REFER: pjsip_xfer_*
     - pjsip-ua regc: pjsip_regc_*
     - pjsip auth: pjsip_auth_*
     - DNS resolver: pj_dns_resolver_*
     - SDP: pjmedia_sdp_*
     - Event subscription: pjsip_evsub_*
   - Do NOT use .opaque_type(".*") — struct fields must be accessible
   - Enable derive_debug(true) and derive_default(true)

5. Create `crates/pjsip-sys/src/lib.rs`:
   - Allow non_upper_case_globals, non_camel_case_types, non_snake_case, dead_code, clippy::all
   - include!(concat!(env!("OUT_DIR"), "/bindings.rs"))

Note: The workspace Cargo.toml uses `members = ["crates/*"]` glob, so pjsip-sys is auto-discovered.
