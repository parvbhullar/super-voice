     Accept stack="pjsip" or "sofia" (backward compat for existing Redis configs)
   - PjsipEndpoint::bridge(&self) -> Option<Arc<PjBridge>>
   - impl SipEndpoint:
     - name() returns &self.name
     - stack() returns "pjsip"

     - stop(): Send Shutdown to bridge, take bridge, set running=false
     - is_running() returns running.load()
