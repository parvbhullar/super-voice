1. **src/handler/endpoints_api.rs** — 5 axum handler functions:
   - `create_endpoint(State(state), Json(body)) -> impl IntoResponse`:
     - Accepts EndpointConfig as JSON body (deserialize with serde)
     - Validates: name non-empty, port > 0, stack is "sofia" or "rsipstack", transport is "udp"/"tcp"/"tls"
     - Calls endpoint_manager.create_endpoint(&config)
     - Also persists to config_store.set_endpoint(&config) for durability
     - Returns 201 with JSON { "name": ..., "status": "running" }
     - On error: 400 for validation, 409 if name exists, 500 for internal errors
   - `list_endpoints(State(state)) -> impl IntoResponse`:
     - Returns 200 with JSON array of { name, stack, listen_addr, status }
   - `get_endpoint(State(state), Path(name)) -> impl IntoResponse`:
     - Returns 200 with full endpoint config + status, or 404
   - `update_endpoint(State(state), Path(name), Json(body)) -> impl IntoResponse`:
     - Stops existing endpoint, updates config in ConfigStore, creates new endpoint with new config
     - Returns 200 with updated config
   - `delete_endpoint(State(state), Path(name)) -> impl IntoResponse`:
     - Stops endpoint, removes from manager, deletes from ConfigStore
     - Returns 204 on success, 404 if not found

2. **src/handler/gateways_api.rs** — 5 axum handler functions:
   - `create_gateway(State(state), Json(body)) -> impl IntoResponse`:
     - Accepts GatewayConfig as JSON body
     - Validates: name non-empty, proxy_addr non-empty, transport is "udp"/"tcp"/"tls"
     - Calls gateway_manager.add_gateway(config)
     - Returns 201 with JSON { name, proxy_addr, transport, status: "active" }
   - `list_gateways(State(state)) -> impl IntoResponse`:
     - Returns 200 with JSON array of GatewayInfo (includes health status from RuntimeState)
   - `get_gateway(State(state), Path(name)) -> impl IntoResponse`:
     - Returns 200 with full gateway config + health status, or 404
   - `update_gateway(State(state), Path(name), Json(body)) -> impl IntoResponse`:
     - Removes old, adds new with updated config
     - Returns 200 with updated config
   - `delete_gateway(State(state), Path(name)) -> impl IntoResponse`:
     - Removes from manager, deletes from ConfigStore
     - Returns 204 on success, 404 if not found

All handlers extract EndpointManager and GatewayManager from AppState. Use `serde_json::json!` for responses. Return proper HTTP status codes and error messages as JSON { "error": "..." }.
