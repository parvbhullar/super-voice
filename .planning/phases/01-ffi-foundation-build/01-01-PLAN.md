Create the `crates/sofia-sip-sys/` directory and crate:

1. **Cargo.toml**:
   - `name = "sofia-sip-sys"`, `version = "0.1.0"`, `edition = "2024"`, `links = "sofia-sip-ua"`
   - `[build-dependencies]`: `bindgen = "0.71"`, `pkg-config = "0.3"`
   - No runtime dependencies (raw FFI crate)

2. **build.rs**:
   - Use `pkg_config::Config::new().atleast_version("1.13.17").probe("sofia-sip-ua")` (note: the actual pkg-config name may be `sofia-sip-ua` or `sofia-sip-ua-glib` â€” try `sofia-sip-ua` first)

   - Use bindgen::Builder to generate bindings from Sofia-SIP headers:
     - Add include path from pkg-config result: `lib.include_paths`
     - Set header content: `#include <sofia-sip/nua.h>\n#include <sofia-sip/sdp.h>\n#include <sofia-sip/su_root.h>\n#include <sofia-sip/nta.h>\n#include <sofia-sip/auth_module.h>`
     - Allowlist functions: `nua_.*`, `su_root_.*`, `sip_.*`, `sdp_.*`, `nta_.*`, `tport_.*`
     - Allowlist types: `nua_.*`, `su_root_t`, `sip_.*`, `sdp_.*`, `nta_.*`, `tport_.*`, `msg_t`, `su_home_t`
     - Set `opaque_type(".*")` for all types (treat as opaque)
     - Generate into `OUT_DIR/bindings.rs`
   - Emit cargo:rustc-link-lib and cargo:rustc-link-search from pkg-config result

3. **src/lib.rs**:
   - `#![allow(non_upper_case_globals, non_camel_case_types, non_snake_case, dead_code)]`
   - `include!(concat!(env!("OUT_DIR"), "/bindings.rs"));`
   - Add a `#[cfg(test)] mod tests` with a basic sanity test that checks a constant or type exists

NOTE: This crate will only compile on systems with Sofia-SIP installed. That is expected â€” it is gated behind the `carrier` feature flag.
