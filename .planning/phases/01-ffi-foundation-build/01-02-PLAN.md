**src/bridge.rs** — `SofiaBridge` struct:
- Fields: `event_rx: tokio::sync::mpsc::UnboundedReceiver<SofiaEvent>`, `cmd_tx: tokio::sync::mpsc::UnboundedSender<SofiaCommand>`, `thread_handle: Option<std::thread::JoinHandle<()>>`
- Constructor `SofiaBridge::start(bind_url: &str) -> Result<Self>`:
  - Create two unbounded mpsc channels: `(event_tx, event_rx)` and `(cmd_tx, cmd_rx)`
  - Spawn a `std::thread::spawn` (NOT tokio spawn_blocking — this is a long-lived thread)
  - Inside the thread:
    a. Create `SuRoot::new()`
    b. Box the `event_tx` and leak it as `*mut c_void` for the C callback `magic` pointer
    c. Call `nua_create(root.as_ptr(), sofia_event_trampoline, magic, NUTAG_URL!(&bind_url), TAG_END!())` — note: NUTAG_URL and TAG_END are Sofia-SIP tag macros; implement as helper functions that produce the correct `tag_type_t`/`tag_value_t` pairs. If bindgen doesn't expose macros, construct the tag structs manually from the exported `nutag_url` and `tag_end` symbols.
    d. Loop: `su_root_step(root, 1)` (1ms timeout) then `cmd_rx.try_recv()` to process commands
    e. On `SofiaCommand::Shutdown`: call `nua_shutdown()`, continue stepping until `nua_r_shutdown` event with status 200, then break
  - Return `SofiaBridge { event_rx, cmd_tx, thread_handle }`
- `pub async fn recv_event(&mut self) -> Option<SofiaEvent>` — wraps `event_rx.recv().await`
- `pub fn send_command(&self, cmd: SofiaCommand) -> Result<()>` — wraps `cmd_tx.send(cmd)`
- `Drop` impl: sends `SofiaCommand::Shutdown`, joins thread
