Create `src/trunk/distribution.rs` implementing gateway selection:

```rust
pub enum DistributionAlgorithm {
    WeightBased,
    RoundRobin,
    HashCallId,
    HashSrcIp,
    HashDestination,
}
```

Implement `select_gateway(algorithm, gateways: &[GatewayRef], context: &SelectionContext) -> Option<&GatewayRef>` where `SelectionContext` holds: `call_id: Option<&str>`, `src_ip: Option<&str>`, `destination: Option<&str>`, `counter: &AtomicU64` (for round-robin).

- **weight_based**: Use weighted random selection. Sum all weights (default weight=1 if None), pick random number in [0, total), iterate and subtract.
- **round_robin**: Use AtomicU64 counter modulo gateway count.
- **hash_callid/hash_src_ip/hash_destination**: Hash the relevant string with a fast hasher (std DefaultHasher), modulo gateway count.

Parse `distribution` string field from TrunkConfig into `DistributionAlgorithm` enum with `from_str` (default to WeightBased for unknown values).

Add DID CRUD to ConfigStore following the exact same pattern as trunk CRUD:
- `set_did(&self, config: &DidConfig)` -- stores with entity type "did", key is config.number. Also tracks engagement: `did:{number}` references `trunk:{config.trunk}`.
- `get_did(&self, number: &str) -> Option<DidConfig>`
- `list_dids() -> Vec<DidConfig>`
- `delete_did(number: &str) -> bool` -- untracks engagement references.

Register `trunk` module in `src/main.rs` or `src/lib.rs` (wherever modules are declared).
